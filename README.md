# Web chat

![alt text](screenshots/main.png "Web Chat")

### Описание

Простой веб-чат. Есть возможность смены аватарки. </br>

### Зависимости

* `express`
* `socket.io`

### Доступные команды

* `npm install` - установить зависимости
* `npm start` - запустить сервер

Чат будет доступен по адресу: localhost:3000
